# Product Definition

## Problem
Riders moving to a new city don't know anyone to ride with.

## Solution
An app to find nearby riders and create ride sessions.

## Target User
- Motorcycle riders
- Expats / travelers
- New riders in a city
- Riders who want to ride with someone else

## MVP Features
- User profile
- Map with nearby riders
- Create ride
- Join ride

## Not in MVP
- Service stations (STO)
- Bike history
- Marketplace
